Comandos Git

git version: indica la version de git instalada

git status

git status -s: muestra los archivos que no se le estan haciendo seguimiento y no se le conoce el estatus y se identifica con los signos de interrogacion

git add + [nombre del archivo es especifico]: coloca el archivo o los archivos al area temporal o stegin area

git add . : coloca todos los archivos en el area temporal o stagin area

git commit -m "texto" : hace referencia al respaldo o toma fotografica del proyecto, se debe detallar en el texto que se esta respaldando.

git log --oneline :

git remote add origin [ruta: https o ssh] / git remote add origin git@github.com:puchenkv/tutorial_git_github.git : establece la conexion con el proyecto, solo debe ejecutarse una sola vez

git push -u origin main
